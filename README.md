# Listas 1, 2 e Exercícios de Maratona de Programação

Este repositório contém a resolução das Listas 1 e 2 da disciplina,
além de exercícios adicionais de Maratona de Programação desenvolvidos em Java.

## 📚 Conteúdo abordado

Os exercícios envolvem:

- Estruturas condicionais
- Estruturas de repetição
- Vetores
- Matrizes
- Manipulação de entrada e saída
- Problemas de lógica e raciocínio algorítmico
- Implementação de soluções com foco em eficiência

## 🧠 Conceitos trabalhados

- Resolução de problemas no padrão de programação competitiva
- Uso de estruturas de dados básicas (arrays e matrizes)
- Controle de fluxo e organização de código
- Aplicação prática de análise de complexidade (O(N), O(N²), O(N×M))

## 🎯 Objetivo

Desenvolver raciocínio lógico e aprimorar habilidades de resolução de problemas,
com foco em desafios acadêmicos e preparação para competições de programação.
